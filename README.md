## container

- コンテナ内の各プロセスは、他所とは隔離されている
- フォアグラウンドでコマンドが終了したらコンテナは停止する
  - たとえば`echo "Hello World!"`させた時と`-d nginx`させた時では、コンテナの生存時間が異なるということか
  - でもバックグラウンドで起動した、フォアグラウンドで動かすnginxが停止したら、そのコンテナは停止してしまう（ややこしい）
